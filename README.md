# thundersec

A theoretical Mozilla Thunderbird add-on that could allow secure communication through client-authoritative protocols.

## What this is about

As a thought experiment, this addon would yield a fully encrypted and secure communication that does not rely on external or nonlocal providers which could be influenced by hostile actors.

## Basic ideas for feature set

It would be freat to be

- Compatible with present-day user certificates
- Fully-local message privacy
